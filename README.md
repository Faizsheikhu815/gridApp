# Real-Time Shared Grid App

A web application where multiple users can interact with a shared grid in real-time. Users can capture tiles, and updates are instantly visible to all connected users.

# Features
- Real-time updates using WebSockets
- Multi-user interaction
- Grid-based UI
- Tile capturing system
- Responsive design

# Tech Stack
- Frontend: React, HTML, CSS
- Backend: Node.js, Express
- Database: MongoDB
- Real-time: Socket.io
